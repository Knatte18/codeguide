# Project Instructions

## Codeguide Routing

ALWAYS READ `_codeguide/Overview.md` BEFORE using any search tool (Grep, Glob, Bash) to find files or symbols. ALWAYS FOLLOW the routing to the correct module doc. NEVER skip this step, even for targeted lookups.

After routing, ALWAYS READ the actual source files before answering factual questions. NEVER answer from `_codeguide/` doc content alone — docs may be stale or incomplete.

## Git

This repo uses a single-branch workflow. When `/mill-commit` is invoked without `--onmain`, treat it as if `--onmain` was passed — commit directly on main.
